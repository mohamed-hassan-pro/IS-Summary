🎯 المفاهيم الأساسية
تعريف System?
Components with boundary work together: Input → Process → Output
تعريف Information System (IS)?
People + Hardware + Software + Networks + Data + Policies
مقارنة IS vs IT?
IS = All components deliver info | IT = HW + SW + Network + Data only
مقارنة Data vs Information?
Data = Raw facts | Information = Meaningful data for users
مقارنة Efficiency vs Effectiveness?
Efficiency = Minimize cost/time | Effectiveness = Support strategy + Increase value
🔄 SDLC - تفاصيل كل Step (مختصر جداً)
مهم جداً SDLC Phases?
Planning → Analysis → Design → Implementation
خطوات PLANNING - السؤال: Why + How?
Step 1: Project Initiation = Identify business value (lower costs ↑ revenues)
Step 2: Project Management = Work plan + Staff + Control techniques
خطوات ANALYSIS - السؤال: Who + What + Where/When?
Step 1: Analysis Strategy = Study current system + Envision new system
Step 2: Requirements Gathering = Develop concept + Build analysis models
Step 3: System Proposal = Present to sponsor → Decision to continue
خطوات DESIGN - السؤال: How will operate?
Step 1: Design Strategy = Company built vs Outsourcing
Step 2: Architecture Design = HW + SW + Network infrastructure
Step 3: Database & File Specs = What data + Where stored
Step 4: Program Design = What programs + What they do
خطوات IMPLEMENTATION - Longest & Most expensive
Step 1: System Construction = Build + Test (performs as designed)
Step 2: Installation = Old OFF + New ON
Step 3: Support Plan = Post-review + Identify changes

📋 System Request & Feasibility
Who writes?
Project Sponsor
مهم Main parts?
1. Business Need (Reason) | 2. Requirements (Capabilities) | 3. Value (Tangible/Intangible benefits) | 4. Special Issues
مهم Feasibility Types?
Technical (Can we build?) + Economic (Should we build?) + Organizational (Will they come?)
Technical Feasibility?
Familiarity (app/tech) + Project size + Compatibility
Economic Feasibility?
Cost-benefit: ROI + BEP + NPV
Organizational Feasibility?
User acceptance + Stakeholders (Champion + Users + Management)
🖥️ Operations Support Systems
مهم OSS Types?
TPS + PCS + ECS
TPS?
Record & process transactions | Sales, Inventory, Accounting
PCS?
Monitor & control physical processes | Sensors | Petroleum refinery
ECS?
Team communications | Email, Videoconferencing
مقارنة Batch vs Online?
Batch = Accumulate & process periodically (Checks at night) | Online = Process immediately (ATM)

DSS?

EIS?

Expert Systems?
Expert advice | Credit advisor
Knowledge Management?
Create + Organize + Disseminate knowledge | Intranet best practices
🏢 IS Classifications
Functional IS?
Support basic functions: Accounting, Finance, Marketing
Cross-Functional IS?
Combination of systems | Support many functions
Strategic IS?
Competitive advantage | Shipment tracking, E-commerce
🌐 E-Business & BPM
E-business?
Internet tech for business processes + E-commerce + Collaboration
BPM?
Improve end-to-end processes | Define → Improve → Eliminate → Adjust workflows
BPA vs BPI vs BPR?
BPA = Tech complement workflows | BPI = Redesign processes | BPR = Complete revamp

💰 Economic Measures
ROI?
(Benefits - Costs) / Costs
BEP?
Costs = Benefits | Net cash flow = 0
NPV?
PV Benefits - PV Costs | PV = Cash/(1+Rate)^n
Development Costs?
Salaries + Consultants + Training + HW/SW + Installation + Data conversion
Operational Costs?
Upgrades + Licensing + Repairs + Salaries + Communications + Training
Tangible Benefits?
Sales ↑ + Staff ↓ + Inventory ↓ + IT costs ↓ + Better prices
Intangible Benefits?
Market share + Brand recognition + Quality + Customer service
