🎯 Introduction
📘 Basic Definitions
تعريف System?
Components with boundary work together: Input → Process → Output
تعريف What are three basic functions of System?
Input (capture elements) → Processing (transform) → Output (transfer results)
تعريف Information System (IS)?
People + Hardware + Software + Networks + Data + Policies
🧩 Components of IS
مهم IS Components?
1. People = End users + Specialists
2. Hardware = Machines + Media
3. Software = Programs + Procedures
4. Data = Facts + Databases
5. Network = Internet + Intranets
عمليات Activities?
Input → Process → Output → Storage → Control
🏢 Classifications of IS by Scope
أنظمة Functional Business Systems?
Focus on operational and managerial applications of basic business functions | Examples: Accounting, Finance, Marketing
أنظمة Strategic Information Systems?
Help get strategic advantage over customers | Examples: Shipment tracking, E-commerce web systems
أنظمة Cross-Functional Information Systems?
Combinations of several IS types | Provide support for many functions
⚖️ Key Comparisons
مقارنة IS vs IT?
IS = All components deliver info | IT = HW + SW + Network + Data only
مقارنة Data vs Information?
Data = Raw facts | Information = Meaningful data for users
مقارنة Efficiency vs Effectiveness?
Efficiency = Minimize cost/time | Effectiveness = Support strategy + Increase value
🔄 Cybernetics
تعريف What is Cybernetics?
System with feedback and control components
مفهوم Feedback?
Data about system performance
مفهوم Control?
Monitoring and evaluating feedback to determine if system is moving toward its goal
🌐 E-Business
تعريف What is E-business?
Use Internet tech to empower business processes, e-commerce, and collaboration
استخدامات How e-business is used?
Reengineer internal processes + Enterprise collaboration + Electronic commerce
🖥️ Operations Support Systems (OSS)
نظام Transaction Processing Systems (TPS)?
Record & process transactions | Sales, Inventory
نظام Process Control Systems (PCS)?
Monitor physical processes | Sensors | Petroleum
نظام Enterprise Collaboration Systems (ECS)?
Team communications | Email, Video
طريقة معالجة Batch Processing?
Accumulate transactions over time and process periodically | Example: Bank processes all checks at night
طريقة معالجة Online Processing?
Process transactions immediately | Example: Bank processes ATM withdrawal immediately
📊 Management Support Systems (MSS)
نظام Management Information Systems (MIS)?
Reports to managers | Daily sales analysis
دعم Decision Support Systems (DSS)?
Interactive decisions | What-if analysis
نظام Executive Information Systems (EIS)?
Critical info executives | Competitors actions
نظام Expert Systems?
Expert advice | Credit application advisor
نظام Knowledge Management Systems?
Create + Organize + Disseminate knowledge | Intranet best practices

🔄 System Development Life Cycle (SDLC)
مهم جداً SDLC Phases?
Planning → Analysis → Design → Implementation
تساؤلات Planning Questions?
Why be built?
How will building?
خطوات Planning Steps?
Step 1: Project Initiation = Identify business value (lower costs ↑ revenues)
Step 2: Project Management = Work plan + Staff + Control techniques
تساؤلات Analysis Questions?
Who use?
What do?
Where & When be used?
خطوات Analysis Steps?
Step 1: Analysis Strategy = Study current system + Envision new system
Step 2: Requirements Gathering = Develop concept + Build analysis models
Step 3: System Proposal = Present to sponsor → Decision to continue
تساؤلات Design Question?
How will operate? (HW, SW, Network, UI, Forms, Reports)
خطوات Design Steps?
Step 1: Design Strategy = Company built vs Outsourcing
Step 2: Architecture Design = HW + SW + Network infrastructure
Step 3: Database & File Specs = What data + Where stored
Step 4: Program Design = What programs + What they do
تساؤلات Implementation Question?
How build, install, support? (Longest & Most expensive)
خطوات IMPLEMENTATION - Longest & Most expensive
Step 1: System Construction = Build + Test (performs as designed)
Step 2: Installation = Old OFF + New ON
Step 3: Support Plan = Post-review + Identify changes
🔄 Business Process Management (BPM)
What is BPM?
Methodology to continuously improve end-to-end business processes
خطوات BPM Process Steps?
1. Define & map steps
2. Improve value-add steps
3. Eliminate non-value steps
4. Adjust workflows class="question-text">BPA vs BPI vs BPR?
BPA (Business Process Automation): Tech complements workflows
BPI (Business Process Improvement): Redesign processes
BPR (Business Process Reengineering): Complete revamp (car to plane)
📋 System Request
مهم Who writes System Request?
Project Sponsor
تعريف What is System Request?
Document describes why build system + expected value
مكون What is Business Need?
Reason for initiating | Examples: Increase sales, Improve market share
مكون What are Business Requirements?
Capabilities system needs | What functions must perform
مهم What is Business Value?
Benefits organization expects
أنواع What are types of Business Value?
Tangible + Intangible
مادي Tangible Value?
Quantifiable | Example: 3% sales increase, Cost reduction
معنوي Intangible Value?
Hard to measure | Example: Customer satisfaction, Brand recognition
مكون What are Special Issues?
Other considerations | Example: Must build before holiday
دور محوري What is Project Sponsor Role?
Primary contact + Ensure direction + Determine business value

تعريف What is Feasibility Analysis?

مهم جداً What are the 3 Areas?

مكون Technical Feasibility?

💰 Economic Measures
مقياس اقتصادي Return on Investment (ROI)?
Rate of return earned | (Total Benefits - Total Costs) ÷ Total Costs
مقياس اقتصادي Break-Even Point (BEP)?
When returns match investment (Year when cumulative cash flow becomes positive)
مقياس اقتصادي Net Present Value (NPV)?
Present value of future cash flows minus initial investment (Adjusted for time value of money)
